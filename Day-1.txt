Assignment 3: Function Design and Modularization

 Factorial Function:
  
    Factorial(n)
      1. Start
      2. if n == 0, then
      2.1. return 1
      else
      2.2. return n * Factorial(n - 1)
      3. End

    Fibonacci Function:
      Fibonacci(n)
       1. Start
       2. if n <= 1, then
       2.1. return n
       else
       2.2. return Fibonacci(n - 1) + Fibonacci(n - 2)
       3. End
